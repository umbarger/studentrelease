About the Student Release System

The Student Release System is designed for Elementary Schools with children from Kindergarten to Fifth grade. The system stores information listed below from each parent or guardian. This allows school administrators to ensure that their students are being picked up by the correct individuals without any doubt. Student safety is dramatically increased as well as parent and guardians have full faith that their child is being picked up by authorized people.

Every vehicle will be assigned and issued a QR code which will be displayed when that vehicle is picking up their child. This allows our system to display when your child is ready to be picked up so the carpool line can run in a more timely manner. This is a time saver for everyone from parents to teachers to students.

Information Stored

  • Family Surname
  • Email Address
  • Phone Number
  • Student First Name
  • Student Last Name
  • Teacher
  • Parent First Name
  • Parent Last Name
  • Parent Mobile Phone Number
  • Vehicle Make
  • Vehicle Model
  • Vehicle Year
  • Vehicle Color
  • Vehicle License Plate Number
  • Vehicle License Plate State
